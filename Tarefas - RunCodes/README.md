# Disciplina Programação Estruturada - IFPI

Este repositório contém as atividades desenvolvidas na disciplina de **Programação Estruturada**, do curso **Técnico em Desenvolvimento de Sistemas** do **Instituto Federal do Piauí (IFPI)**, ministrada pelo **Professor Ritomar Torquato**. O foco da disciplina foi fornecer uma base sólida em **lógica de programação** e no uso de **Python** para resolver problemas computacionais.

## Sobre a Disciplina

A disciplina de **Programação Estruturada** teve como objetivo introduzir os alunos aos conceitos fundamentais da programação, utilizando **Python** como linguagem principal. Ao longo do curso, foram trabalhados os seguintes tópicos:

- **Lógica de Programação**: Estruturas de controle, tomadas de decisão e fluxo de execução.
- **Variáveis e Tipos de Dados**: Definição e manipulação de variáveis em Python.
- **Listas e Dicionários**: Estruturas de dados essenciais para armazenar e manipular coleções de informações.

Esses conceitos foram aplicados para resolver problemas práticos e exercitar a lógica de desenvolvimento de soluções computacionais.

## Estrutura do Repositório

Este repositório está organizado da seguinte forma:

- **[Sem01](./Sem01)**: Contém as atividades e exercícios realizados durante a primeira semana do curso.
- **[Sem02](./Sem02)**: Atividades desenvolvidas na segunda semana
- **[Sem03](./Sem03)**: Exercícios da terceira semana
- **E assim por diante...**

Cada pasta semanal contém as atividades práticas desenvolvidas durante aquela semana, com os exercícios de programação que ajudam a reforçar os conceitos aprendidos.

## Tecnologias e Linguagens Utilizadas

Durante a disciplina de **Programação Estruturada**, foram utilizadas as seguintes tecnologias e linguagens:

- **Linguagem de Programação**: Python
- **Ambiente de Desenvolvimento**: Visual Studio Code
- **Estruturas de Dados**: Variáveis, Listas, Dicionários
